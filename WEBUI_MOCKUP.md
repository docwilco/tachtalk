## Color Scheme

The web UI uses a dark theme optimized for automotive/night use:

- **Background**: #1a1a1a (Dark gray)
- **Text**: #ffffff (White)
- **Accent**: #00ff00 (Green)
- **Panels**: #2a2a2a (Light gray)
- **Blink Panel**: #3a2a2a (Reddish tint)
- **Buttons**: #00ff00 (Green with black text)
